# 🚀 Jarvis Term: The Ultimate AI Engineering Console (v0.1.6)

Jarvis Term is a high-performance, standalone terminal emulator built for the modern engineer. Inspired by **Warp** and powered by the **JARVIS AI Assistant**, it combines native speed with deep agentic intelligence. 

## 💎 What's New in v0.1.5
- **Gemini-Style UI Blocks:** Transparent reasoning panels and dedicated tool execution boxes.
- **Strategic Plan Mode:** Design roadmaps before execution with the new `/plan` interface.
- **Ollama Account Connectivity:** Sync with Ollama Cloud and managed model libraries.
- **Hyper-Automated Setup:** Zero-config deployment with "Automation Mode."
- **Multi-Model Support:** Native integration for **DeepSeek**, **Kimi**, **Qwen**, and **Gemma**.

## ⚡️ Key Features

### Warp-Style Command Interface
- **Intelligent Command Bar:** Dedicated input bar for commands, chat, and agent triggers.
- **Multi-Session Power:** Manage independent workspaces in a single tabbed window.
- **Keyboard-First Focus:** Seamless navigation via `Tab` and `Shift+Tab`.

### 🧠 Integrated JARVIS AI Agent
- **Native Assistant:** Every session automatically initializes the JARVIS sovereign brain.
- **Autonomous Debugging:** Launch repair loops directly from your console.
- **Multi-Model Aggregator:** Switch between 10+ local and cloud AI providers.

### 🏢 Professional Workflow Tools
- **Integrated Document Viewer:** Side-pane side-by-side reading of source code and docs.
- **Drag-and-Drop Analysis:** Instantly audit any file by dropping it onto the console.
- **Native macOS Parity:** Full system menu support and native shortcuts.

## 📦 Installation

### Homebrew (macOS)
```bash
brew install --cask https://raw.githubusercontent.com/nave433-blip/jarvis-term/master/jarvis-term.rb
```

### Manual Build
1. Clone repo: `git clone https://github.com/nave433-blip/jarvis-term.git`
2. Run build: `python3 build.py`
